# ionicv2Github
Sample github app for Ionic v2 test development

```$ git clone```

```$ npm install -g cordova```
```$ npm install -g ionic@beta```
```$ npm install -g typescript```
```$ npm install -g typings```

```$ ionic serve```

```$ npm install -g ios-deploy```
```$ npm install -g ios-sim version```
```$ ionic build ios```
```$ ionic run ios```
